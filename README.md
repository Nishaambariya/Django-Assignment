# Todo WebApp

ToDo WebApp is an Django Project used to store our todo list.



## Run Locally

Clone the project

```bash
  git clone https://github.com/Nishaambariya/Django-Assignment.git
```

Go to the project directory

```bash
  cd todo-DjangoApp
```

Start the server

```bash
  python manage.py runserver
```
